How to Make a Personal Web Page on GitHub!

If you haven't already done so, create an account on GitHub. Then, follow the step-by-step directions below!

  1. Login to GitHub and select 'Repositories.'
  2. Select 'New' to create a new repository where you'll put the code for your webpage.
  3. Create a name for your repository in the following format:
    "username".github.io
    Then create your repository.
  4. In a code editor of your choice, such as Visual Studio Code, create a basic HTML file with anything you'd like to include in your webpage.
  5. Save the file and then head back to your repository and find where it says "uploading an existing file." Select this and you should see a space to drop files.
  6. Drag your html file and any other attached files, such as images, into your repository.
  7. Commit your changes and test out your new webpage in a browser of your choice!
